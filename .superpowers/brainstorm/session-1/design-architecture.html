设计 A：总体架构

单文件网页应用，浏览器直连 Azure Speech

系统边界
朗读练习网页
HTML + CSS + JS（单文件）
↕ 浏览器内录音 + HTTPS
Azure Speech Services
发音评估 API

网页内部模块

  • 配置区：Azure region + key + 语言（默认 en-US）
  • 文本区：粘贴原文
  • 录音控制：开始 / 停止 / 重录
  • 结果展示：原文高亮（绿/黄/红）
  • drill 面板：切换到问题句 / 问题词循环练习

核心原则：无后端、无构建步骤、双击 HTML 即可运行。Azure Key 由用户在页面内填写，只在本地测试使用。
